# ESP32-S3 Audio Spectrum Visualizer (FFT)

Визуализатор аудиоспектра на базе **ESP32-S3**, использующий аппаратный ADC (DMA) и 8-битную параллельную шину для дисплея **ILI9486**.

## 🛠 Железо
* **MCU:** ESP32-S3 (DevKitC-1)
* **Дисплей:** ILI9486 (3.5" LCD, 8-bit Parallel)
* **Микрофон:** Аналоговый (подключение к ADC1_CH0 / GPIO 1)
* **ЦАП:**  PCM5102A пины:

    BCK -> GPIO 14
    DIN -> GPIO 13
    LCK -> GPIO 12
    I-Выход (L-Out): На вход I внешнего смесителя.
    Q-Выход (R-Out): На вход Q внешнего смесителя.
    
## 🔌 Подключение (8-bit Parallel)

| Сигнал дисплея | GPIO ESP32-S3 | Примечание |
| :--- | :--- | :--- |
| **D0 - D3** | 39, 40, 41, 42 | Шина данных (Low) |
| **D4 - D7** | 45, 46, 47, 48 | Шина данных (High) |
| **WR** | 4 | Запись (Write) |
| **RS (DC)** | 6 | Команда / Данные |
| **CS** | 7 | Chip Select |
| **RST** | 15 | Reset |
| **RD** | **3.3V** | Соединено с 3.3V (защита!) |
| **VCC / LED** | **5V** | Через перемычку VBUS-VIN |

## 🚀 Особенности реализации
1. **ADC Continuous Mode (DMA):** Стабильная частота дискретизации 44.1 кГц.
2. **FFT:** 1024 точки (библиотека `esp-dsp`).
3. **Canvas Drawing:** Отрисовка спектра (до 256 бинов) целым кадром через DMA-буфер для минимизации мерцания.
4. **DC Offset Removal:** Программное вычитание среднего значения для удаления постоянной составляющей сигнала.
